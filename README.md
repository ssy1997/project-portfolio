# Project Portfolio

这里整理了我近期的几个 AI / Agent 相关项目，方便集中查看。

## Projects

### QuestFlow

Coze 工作流集合，围绕题库生成与题目审核两个环节，适合搭建“生成 + 质检”的题库流程。

- GitHub: https://github.com/ssy1997/QuestFlow

### survey-agent

面向调研与信息采集场景的 Agent 项目，用于提升问卷、信息整理和分析相关流程的自动化效率。

- GitHub: https://github.com/ssy1997/survey-agent

### ai-ootd

AI 穿搭相关项目，聚焦服饰推荐、搭配生成或相关智能交互体验。

- GitHub: https://github.com/ssy1997/ai-ootd

### literature-retrieval-skill

文献检索技能项目，面向学术搜索、资料获取与论文调研等场景，强调检索效率和结构化整理。

- GitHub: https://github.com/ssy1997/literature-retrieval-skill

## Focus

这些项目主要集中在几个方向：

- AI Agent 与工作流自动化
- 信息检索与知识处理
- 垂直场景产品化尝试
